# 25 + 5 Clock ⏱️

A **Pomodoro-style timer** application built with React.  
This project is part of the **freeCodeCamp Front-End Development Libraries Certification**.

---

## 🚀 Features
- **25-minute work session** and **5-minute break** (default).
- Adjustable **session length** and **break length**.
- **Start / Stop** functionality to control the timer.
- **Reset** button to restore default settings.
- Plays an **audio beep** when switching between session and break.
- Fully responsive design for desktop and mobile.

---

## 🛠️ Tech Stack
- **React** (functional components + hooks)
- **JavaScript (ES6+)**
- **HTML5 / CSS3**
- FreeCodeCamp project requirements
